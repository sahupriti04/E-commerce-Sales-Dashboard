# E-commerce Sales Dashboard

A 4-week Data Science internship project showcasing data cleaning, exploratory data analysis, machine learning, visualization, and real-world analytics using Python.

---

# Project Overview

This repository contains weekly internship tasks focused on developing practical Data Science and Machine Learning skills.

The projects cover:
- Data Cleaning & Preprocessing
- Data Visualization
- Exploratory Data Analysis (EDA)
- Predictive Modeling
- Real-world Retail Analytics

---

# Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook / Google Colab

---

# Repository Structure

| Week | Project | Skills Learned |
|------|----------|----------------|
| Week 1 | Data Cleaning & Visualization | Data preprocessing, charts, outlier detection |
| Week 2 | Predictive Modeling | Machine learning, regression, model evaluation |
| Week 3 | Exploratory Data Analysis | Correlation analysis, statistical insights |
| Week 4 | Real-world Retail Analytics | End-to-end analytics workflow |
